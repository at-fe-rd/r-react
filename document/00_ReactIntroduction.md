# Giới thiệu về React


#### Xu hướng Single page
React (hay còn gọi là React.js) là một thư viện JavaScript với xu hướng Single Page Application.

#### Sử dụng thư viện JS khác
Khác với các thư viện khác đang hướng đến một mô hình MVC thì React nổi bật lên với sự đơn giản, dễ sử dụng và dễ phối hợp với các thư viện JavaScript khác.

#### Chia nhỏ thành các thành phần
React hỗ trợ việc xây dựng những thành phần UI (component) có tính tương tác cao, có trạng thái và có thể sử dụng lại.

#### Chỉ thay đổi thành phần
React sử dụng khái niệm DOM ảo (Virtual DOM) để chọn lựa và render những phần tử cần thiết, chọn lựa những component cần thiết để thay đổi ít nhất có thể.


#### Multi flatform
Hiện nay React không chỉ dùng để viết web mà còn dùng để viết Application trên Android, IOS và các flatform khác
- React và React DOM: xây dựng web browser
- React và React Native: xây dựng application trên các thiết bị di động

Ngoài ra, cũng có thể xây dựng Web browser bằng React Native, tuy nhiên hiện nay các Core của React Native chỉ đáp ứng được một phần nhỏ để xây dựng Web UI.
